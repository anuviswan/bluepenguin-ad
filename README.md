# Blue Penguin Admin Panel

Admin Panel for Blue Penguin, the internal system used to manage products, categories, pricing, and inventory for the Blue Penguin jewelry brand.
This application is designed to provide a clean, efficient workflow for managing handcrafted jewelry products and related data.
